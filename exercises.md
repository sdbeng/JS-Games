#Exercises
##1 Looping a triangle

Write a loop that makes seven calls to console.log to output the following triangle:
```
#
##
###
####
#####
######
#######
```
my_solution:
```
var t = 6;
var hash = "";
for(var i=0; i<=t; i++){
  hash = hash + "#";
  console.log(hash);
}
```
